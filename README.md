# Sushi Website
================

## Getting Started
---------------

To run the application, follow these steps:

### Install Dependencies
```bash
npm install
```

### Run the Application
```bash
npm run dev
```
This will start the development server and make the application available at `http://localhost:5173`.

### Accessing the Website
To access the sushi website, use the following credentials:

Username: `admin`
Password: `password`
Note: Please use these credentials only for testing and development purposes.
